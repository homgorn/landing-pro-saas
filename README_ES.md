## 🛠 Características del Constructor Universal

### 🦾 Orquestación Multi-Agente
El sistema utiliza un ciclo de 5 etapas para garantizar la calidad:
1. **Vision Analysis:** Extrae el **Design DNA** (paleta, fuentes, cuadrícula) de una captura de pantalla.
2. **Dynamic Research:** Utiliza la **API de Brave Search** para sintetizar tendencias y claves LSI.
3. **Persuasive Copywriting:** Redacción impulsada por IA adaptada a su nicho e instrucciones.
4. **Adaptive Code Gen:** Genera código semántico en HTML5/CSS3 puro sin dependencias.
5. **Deep QA Audit:** Verificación automática de adaptabilidad móvil e integridad.
