## 🛠 Возможности универсального конструктора

### 🦾 Мультиагентный Pipeline
Система использует 5-стадийный цикл для обеспечения качества:
1. **Vision Analysis:** Извлекает **Design DNA** (палитру, шрифты, сетку) из скриншота-донора.
2. **Dynamic Research:** Использует **Brave Search API** для синтеза актуальных трендов и LSI-ключей.
3. **Persuasive Copywriting:** ИИ-копирайтинг, адаптированный под вашу нишу и инструкции.
4. **Adaptive Code Gen:** Выдает семантичный код на чистом HTML5/CSS3 без лишних библиотек.
5. **Deep QA Audit:** Автоматическая проверка мобильной адаптивности и соответствия ТЗ.
