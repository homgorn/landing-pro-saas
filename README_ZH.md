## 🛠 通用构造器核心特性

### 🦾 多智能体流水线编排
系统采用 5 阶段循环以确保卓越品质：
1. **视觉分析 (Vision):** 从参考截图中提取 **Design DNA**（配色、字体、布局网格）。
2. **动态调研:** 调用 **Brave Search API** 合成实时市场趋势和 LSI 关键词。
3. **说服力文案:** 结合行业深度和用户指令的 AI 文案。
4. **自适应代码生成:** 输出纯净的 HTML5/CSS3 语义化代码，无多余依赖。
5. **深度 QA 审计:** 自动验证移动端自适应及业务逻辑完整性。
